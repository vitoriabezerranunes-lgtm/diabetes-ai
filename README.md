# diabetes-ai

Diabetes risk prediction using XGBoost, trained on the Pima Indians Diabetes dataset.

## Stack

- XGBoost
- scikit-learn
- Streamlit
- pandas

## Usage

```bash
pip install -r requirements.txt
python download_data.py
python train.py
streamlit run app.py
```

## Dataset

768 patient records, 8 clinical features. Source: UCI Machine Learning Repository.
